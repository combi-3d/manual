    RAW to JPEG conversion/RAW現像

    Applications

    Use one of the apps below. In 2014, the beginning of CoMBI development, Adobe Bridge was the only app that could stably convert more than 1000 images. I'm using Bridge since then. Currently, all apps below works well.

    下記のいずれかを使用します。2014年頃は、Adobe Bridgeだけが安定していたので、以来、Bridgeを使っています。いまでは、カメラ会社のカメラ付属アプリでも安定して行えます。

    • Adobe Bridge
    • Adone Lightroom
    • Nikon Capture NX-D, NX studio (2021.3-) (free)
    • Sony Imaging Edge (free)
    • Canon Digital Photo Professional (free)
    • Darktable (free)

    Procedures/手順

    Adobe Bridge

    1. Select an image and right click. Open in Camera RAW
    2. Adjust brightness, contrast, color balance etc.
    3. Crop to exclude needless region. Keep ｃircumference of the block.
    4. Close Camera RAW by clicking Done.
    5. Right click on the adjusted image. Develop settings> Copy setting
    6. Select ALL images and right-crick, Develop settings> Paste setting
    7. Check all, OK
    8. Convert to JPEG. select all, drag and drop to Export tag in the left window, then convert.
    1. 一枚画像を選んで、右クリック、Camera RAWで開く
    2. 明るさ、コントラスト、カラーなどを調整する。
    3. クロップして、ブロック外の不要な領域を除く。ブロック外周は残すこと。
    4. 完了を押して、Camera RAWをとじる
    5. 調整した一枚の上で、右クリック、Develop settings> Copy setting
    6. すべての画像を選んで、右クリック、Develop settings> Paste setting
    7. Check all, OK
    8. JPEGに書き出す。select all, 左のウィンドウのタグへドラッグする。convert

    Sony Imaging Edge

    1. "Edit" Adjust an image, then save setting file.
      1. Edit> Image Processing Settings, Save...
    2. "Viewer"
      1. Select all images
      2. Export button or File> Export...
      3. 任意の画像処理設定をすべてに適用
      4. 保存しておいた設定ファイルを選択する
    3. "Edit"
      1. 画像処理設定を適用したい画像を表示
      2. 編集> 画像処理設定> 読み込み（適用）
      3. 設定ファイルを選択し、開く
    1. Editで画像調製したのち、設定ファイルを保存する
      1. 編集> 画像処理設定, 保存
    2. Viewerの場合
      1. 連続画像を選択、
      2. Exportボタン、または、ファイル> 現像出力
      3. 任意の画像処理設定をすべてに適用
      4. 保存しておいた設定ファイルを選択する
    3. Editの場合
      1. 画像処理設定を適用したい画像を表示
      2. 編集> 画像処理設定> 読み込み（適用）
      3. 設定ファイルを選択し、開く

    Tips

    • CPUが働きます。
    • メモリを喰います。約100枚変換すると1GBうまります。開放されないのでメモリがいっぱいになると、変換作業が遅くなります。
    • 設定ファイルの扱いがアプリによって異なります。多数の画像を一括変換する場合、Bridgeでは、現像設定ファイルが、各RAWにひとつづつできます。一方、Sony Imaging Edgeでは、設定ファイルがひとつできると、それを呼び出しながら、JPEG変換＆保存をします。生成される設定ファイルはひとつだけで済みます。
