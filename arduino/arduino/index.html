    Using an ATmega328P standalone.

    Reference, official

    From Arduino to a Microcontroller on a Breadboard, 2018

    Reference in Japanease

    Voltechno.com/壊れたArduinoを修理！ブートローダーの書き換えで修理する方法, 2018-2021

    音遊び日記/atmega328にブートローダとスケッチを書き込む, 2018

    ネット上には情報はたくさんあります。新しいArduino IDE 1.8.x(2021時点)の記事がよいと思います。公式ページには、1.0.x用の方法と、それ以外(1.6, 1.8)用の方法が記載されています。

    Preparation

    • Arduino Uno
    • Bootloader writing circuit (a 16 MHz crystal, a 10k resistor, and two 18 to 22 picofarad (ceramic) capacitors, a zero-pressure 28-pin socket)
    • USB-serial convirsion cable
    • Sketch writing circuit with USB-serial converter (USB-serial converter, a zero-pressure 28-pin socket)
    • Jumper wire
    • Computer

    Don't forget !

    Install a driver for USB-serial converter. Download site (FTDI):

    ftdichip.com, Drivers

    Reference in Japanease (Switch Science):

    Switch-science.com, magazine 2013/01/23

    Procedure

    Make Arduino Uno as Arduino ISP (in-system programmer)

    1. Open the sketch File > Examples > 11. ArduinoISP.
    2. Connect Arduino Uno with PC using USB cable.

    3. Upload the sketch to Aruduino Uno with the settings as usual;

      • Tools > Board: Arduino Uno
      • Tools > Port: /dev/cu.usbserial-A104UQHR
      • Tools > Programmer: AVRISPmkII

      You will find Done Uploading

    Burn bootloader on ATmega328P using Arduino ISP

    1. Connect Arduino Uno (D10-13, 5V, GND) to an ATmega328P on the bootloader writer circuit.

      • Tools > Board: Arduino Duemilanove or Nano w/ ATmega328 公式サイトには、「もしくはATmega328 on a breadboard (8 MHz internal clock) 水晶発振子を使わない場合」とある。たぶんまえもって、board情報をダウンロードし、ArduinoフォルダにHardwareフォルダをつくって格納しておくやつ。)
      • Tools > Port: /dev/cu.usbserial-A104UQHR
      • Tools > Programmer: Arduino as ISP

    2. Tools > Burn bootloader. You will find Done Burning bootloader

    3. Remove ATmega328P from the bootloader writer circuit.

    Upload a sketch to ATmega328P via USB-serial converter.

    There is anothor way using USB-serial convert function on Arduino Uno board. However, ATmega328P chip on Arduino needs to be removed to use such function. Since I'm not confident that I can repeat the removal and insertion without damaging the ATmega pins, I bought USB-serial converter from Switch Science. Installation of a driver (FDTI) is needed before use.

    公式サイトには「Arduino UnoボードのUSB-シリアル変換機能」を利用する方法が紹介されています。しかし、この機能を使うためには、ArduinoのATmega328Pチップを取り外す必要があるそうです（種々のネット情報では、外さずにスケッチ書き込みまでやる記載もあるけれど）。私は、ATmegaのピンを傷つけずに抜き差しを繰り返す自信がないので、スイッチサイエンスの「USB-serial converter」を購入しました。使用するにはドライバ（FDTI社）のインストールが必要です。

    1. Connect the sketch writer circuit to PC via USB-serial converter.
    2. Put ATmega328P onto the sketch writer circuit.
    3. Open the sketch to be uploaded.

    4. Upload with the settings below;

      • Tools > Board: Arduino Pro or Pro mini
      • Tools > Port: /dev/cu.usbserial-A104UQHR
      • Tools > Processor: ATmega328P(3.3V, 8MHz)
      • Tools > Programmer: AVRISPmkII or Arduino as ISP　（関係ない？）

    5. You will find Done Uploading
