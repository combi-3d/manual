    Arduino micro tips

    231020、240211、240217 HPへ

    Change bootloader

    Arduino microの公式デフォルトブートローダでは、リセットに８秒かかる。BootloaderとUserCodeの両方を読み直すため。 Spatkfun社から、改良版のBootloaderが配布されている。リセットボタンを押す（シングルクリックする）と、UserCodeだけ読み直し、0.75secで再起動できる。ダブルクリックでbootloaderから読み直す。 Arduino micro、SparkFun Pro microは互換性があるので書き換えられる。

    準備

    マイコンATmega32U4には2種類存在する。Arduino microは5V,16Mhzである Arduino IDEにSparkFun製品のライブラリをインストールする 書き込み装置にしたてるArduino、私はUNOを用意した 書き換えられるArduino micro ジャンパーケーブル　オスーメス　6本

    作業

    1. Sparkfunのファイルをインストールする

      Arduino IDE > Settings

      https://raw.githubusercontent.com/sparkfun/Arduino_Boards/master/IDE_Board_Manager/package_sparkfun_index.json

      Tools >Boards >nBoards Manager…

      SparkFun AVR Baordsをinstall 1.1.13 (2023)

      Tools > Boardsに、Sparkfun AVR Boardsが表示される

    2. UNOを書き込み装置に仕立てる　Arduino as ISPの準備

      Arduino UNOをUSBでPCに接続する File > Examples > 11. ArduinoISP > ArduinoISP 書き込む

    3. 配線する

      Arduino UNO 5V, GND, 10,11,12,13 Arduino micro ICSPに。

    4. 設定する

      Tools > Board: “SparkFun pro micro”　（ボード） Tools > Port: “/dev/cu.usb…”　（ポート） Tools > Processor: “ATmega32U4 (5V, 16MHz)　（プロセッサー） Tools > Programer: “Arduino as ISP” （書き込み装置）

      注意：Arduino microのATmega32U4は、5V 16MHzである

    5. ブートローダを書き込む

      Tools > Burn bootloader（ブートローダを書き込む） RESETが一瞬で完了する。8秒かからなくなった。ダブルクリックで8秒。 RX・TXのLEDが常時点灯するようになる
