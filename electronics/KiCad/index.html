    KiCad

    KiCad is a donation-ware to design circuit and PCB. KiCad was created in 1992 by Jean-Pierre Charras in France. In 2013 the CERN started contributing to improve KiCad. KiCad is compatible with Win, Mac and Linux.

    (2023 KiCAD7 install, 2024.2.23-25 KiCAD7.0.8)

    2021年からKiCadでCoMBIの回路やプリント基板をデザインしています。それ以前はCADLUS（Winのみ）を使っていましたが、私がWindows環境から離れつつあったため、KiCadへ移行しました。KiCadは1992年にフランスの研究者によってリリースされ、改良が続けられています。2013年からは欧州原子核研究機構が開発に参加するようになり、改良がより進むようになりました。KiCadは寄付または無料で利用できます。Win Mac Linuxに対応。2021年v6から日本語に対応。

    Install and prepare library/インストール

    (2023 KiCAD7)

    1. Official site. https://www.kicad.org
    2. "DOWNLOAD" tab to get an installer. Install KiCad7(2023).
    3. (Parts library is included in ver.6 or newer.)
    4. Additional library DigiKey JST's connectors
    5. New Project or New Project from Template (Arduino micro etc.)

    部品のシンボルとフットプリントを探してダウンロード、KiCADサイトからでも。今回はArduinoを追加した。Digikeyもいれたけど使わず。JSTのコネクタは元々入っていた v7

    回路図エディタで、設定（メニューバーの右から２つめ）＞　シンボルラブラリを管理　＞　フォルダマークから、Openできるフォルダやファイルならいくつでもやっておく

    PCBエディタで、設定（メニューバーの右から２つめ）＞　フットプリントラブラリを管理　で同様に

    New project/プロジェクトを新規作成

    1. Two files will be appeared within a project; .kicad_pcb and .kicad_sch

    プロジェクト内にファイルが２つ、回路図と基板

    Curcuit/回路図をつくる

    1. 回路図上の部品記号は「シンボル」、PCB上で部品の形状は「フットプリント」と呼ぶ。
    2. フットプリントエディタ：ボタン「フットプリントを作成、削除、編集」より。個々の部品のフットプリントを一つづつ設定する
    3. ボタン「フットプリント割り当てツール」では、部品一覧が出てくる。こちらのほうが便利。
    4. 前項でのシンボルとフットプリントのライブラリを揃えておくこと

    PCB, parts and circuit/基板をデザインする

    1. PCBエディタを起動
    2. ボタン「回路図で行われた更新で基板を更新」
    3. すべての部品が現れる
    4. 配置する
    5. 配線する　表はF.Cu、裏はB.Cu　線幅は1mm/1A、シグナルは0.15〜0.5らしい。銅箔はデフォルト0.035 mm
    6. シルク 表はF.SiikScreen、裏はB.SilkScreen
    7. ベタ（まだやったことない）

    Outline/基板外形

    1. 線を引く、レイヤーはEdge.cut
    2. 線や孤を組み合わせて複雑な形状をつくれる。端点をきっちり位置合わせする

    Mount hole/PCBにネジ穴

    1. ボタン「モジュールの追加」
    2. MountingHole_3.2mm_M3
    3. OK

    Check/点検

    1. PCBで配線したら、ボタン「デザインルールチェッカー」（DRC）
    2. 近すぎる配線、未接続の端子、外形の線が微妙にくっついていない
    3. エラーが無くなるまで繰り返す

    3D

    1. 表示 > 3D viewer

    Logo as a silkscreen/ロゴを基板に印刷する

    Design by Illustrator etc

    1. デザインする　大きさはのちのKiCADイメージコンバーターで決める
    2. Export as…
    3. PNG、300 dpi、今回はPNG, Artboardごと JPG, BMPでもよいらしい
    4. 今回は高さ10mm。文字の線幅がちいさいかも。Photoshopで選択して、丸くsmooth, 1 pix拡張 expand
    5. 文字を大きくしてデザインし直した。
    6. QRコードを生成する
    7. Adobe Express（オンライン、登録なし・無料で使える範囲）、InDesign, Illustrator

    P板.comの規格

    • 推奨基準 線幅 0.15mm 以上、文字高さ 1.5mm 以上。
    • 最小基準 線幅 0.127mm、高さ 1.0mm。それ以下はカスレやにじみなどで判読不可能となる可能性あり
    • 印刷位置ずれ±0.25mm は許容差

    8-bit binary, ImageJ

    8-bit、二値化した画像がよい。KiCADイメージコンバータでの失敗がすくないらしい。 1. Image > Type > 8 bit 1. Image > Adjust > Threshold * 128-255 * Default * B&W * Apply、0と255で構成されていることを確認する（反転flipは必要ない） 1. File > Save as…

    8-nit binary, Photoshop

    1. Image > Mode > 8-bit
    2. Image > Adjustments > Threshold

    KiCAD イメージコンバーター

    1. デザインした画像を読み込む
    2. 表面シルクスクリーンにチェック
    3. フットプリント（KiCAD.modファイル）にチェック
    4. negaをチェックしたら、白い領域（255）がシルクになる
    5. チェックしないと、黒い領域（0）がシルクになる（今回はこれ）
    6. 余計な領域が含まれていてもシルクにならない色であれば無視されるが、前もってクロップしたほうが良く、
    7. ここでサイズを決めるため。今回は高さ12 mm。のちのPCBではサイズ変えられない。
    8. ファイルを保存

    Finderでファイル操作（macの場合）

    1. /Applications/KiCad/KiCad.app/Contents/SharedSupport/footprints/Symbol.pretty/
    2. ここにつくったファイルを保存する、Finderでファイル操作

    KiCAD PCBエディタ

    1. ボタン「フットプリントを追加」、フットプリントを選択。ファイル名を検索すると見つけやすい
    2. フットプリントのプロパティ（ダブルクリック）で、
      • 位置 > 面 > 裏面
      • レイヤー > B.Silkscreen
